My Life Hack App
________________________________________
1. Introduction
My Life Hack App is a simple Android mobile application built using Kotlin in Android Studio. The app is designed as a quiz game where users must decide whether a statement is a real life hack or a myth.
The app helps users learn how to identify useful information and avoid believing false internet trends.
________________________________________
2. Problem Statement
Many people find it difficult to know which online life hacks are real and which ones are fake. Social media contains many viral tricks that are not always true.
This creates confusion and can lead to people using unsafe or useless advice.
________________________________________
3. Aim of the Project
The aim of this project is to:
•	Create a simple quiz app using Kotlin 
•	Help users learn real life hacks 
•	Teach users to identify myths 
•	Make learning fun and interactive 
________________________________________
4. Objectives
The objectives of the app are:
•	To display questions one at a time 
•	To allow users to answer True or False 
•	To give instant feedback (Correct or Wrong) 
•	To track the user’s score 
•	To move to the next question using a button 
________________________________________
5. Tools and Technologies Used
The following tools were used to develop the application:
•	Android Studio (Development environment) 
•	Kotlin (Programming language) 
•	XML (User interface design) 
•	Android SDK (Software development kit) 
________________________________________
6. System Design
The application is divided into two main screens:
6.1 Main Screen
•	Contains a Start button 
•	Takes the user to the quiz screen 
6.2 Quiz Screen
•	Displays questions 
•	Contains True and False buttons 
•	Shows feedback after each answer 
•	Displays score updates 
________________________________________
7. How the App Works
1.	The user opens the app 
2.	The main screen appears 
3.	The user clicks the Start button 
4.	A quiz question is displayed 
5.	The user selects True or False 
6.	The app checks the answer 
7.	Feedback is shown (Correct or Wrong) 
8.	The user clicks Next for another question 
9.	The score increases for correct answers 
________________________________________
8. Features of the App
•	Simple and user-friendly interface 
•	True or False quiz format 
•	Instant answer feedback 
•	Score tracking system 
•	Easy navigation between questions 
________________________________________
9. Challenges Faced
During development, the following challenges were faced:
•	XML errors that caused build failures 
•	Resource compilation errors in Android Studio 
•	Connecting buttons correctly to Kotlin code 
•	Ensuring the app does not crash when switching questions 
These problems were solved by debugging and fixing XML and Kotlin code step by step.
